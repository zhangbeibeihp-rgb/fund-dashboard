📋 选基方法论说明

1. 收益风险类因子(25%)

2. 回撤控制因子(15%)
→ 最大回撤、回撤修复时间、下行捕获率
3. 规模流动性因子(15%)
→ 基金规模(＞10亿)、日均成交额、机构持有比例
4. 费率成本因子(15%)
→ 管理费、托管费、申购费、赎回费阶梯
5. 晨星评级因子(10%)
→ 3/5/10年MRAR星级(5星＞4星＞3星)
6. 基金经理因子(10%)
→ 管理年限(＞3年)、历史业绩稳定性、风格一致性
7. 跟踪误差因子(指数型)
→ 年化跟踪误差＜0.5%、信息比率
8. 隐形交易能力因子
→ 净值回归残差、波段交易能力
